Throw `EndpointError` subclasses in handlers wrapped by `withApiHandler()`.

```ts
import { withApiHandler, UnauthorizedError, NotFoundError } from "@ideative/next-handler";

export const GET = withApiHandler(async (req) => {
  if (!req.headers.get("authorization")) throw new UnauthorizedError("Missing token");
  throw new NotFoundError("User");
});
```

The wrapper serializes known errors with status codes (`401`, `404`, etc.) so your API wire format stays consistent.
